# Генератор Тестовых Данных

Streamlit приложение для генерации тестовых данных для QA-специалистов.

## Функции
- Генерация персональных данных, контактов, документов РФ
- Валидация СНИЛС, ИНН, банковских карт
- Экспорт в CSV, JSON, XML, SQL
- Связанные данные (пользователи и заказы)

## Запуск
pip install -r requirements.txt
streamlit run app.py

Приложение размещено на Railway. .
